# LLM Freeway

An Opensource proxy server for litellm

## features

IOC JWT based auth

* `/chat/completions`
  * restrict model access
  * restrict spend
  * rate limiting

* `/spend/logs`
  * see what you, and others in your team have spent

* `/user/`
  * get your own user details or others if youre an admin
  * create/update
